# standard

> 规范文档模板。文件名：`standard-<主题>-<行为>.md`

**区别于** guide：standard 是约束规则，guide 是操作步骤。standard 说"必须怎样"，guide 说"怎么做"。

**必填节**：
- **引言** — 受众、目标、范围
- **理念** — 核心原则（不超过 5 条）
- **规则** — 按域分组，每条含编号、标题、理由、示例（没有示例的规则无法判断对错）

**可选节**：
- 自动化检查 — 何时加：有 lint/CI 可执行
- 例外情况 — 何时加：存在已知合理例外

**常见错误**：
- ❌ 规则没有示例（无法判断对错）
- ❌ 理念超过 5 条（失去聚焦）
- ❌ 规则写"应该"而不是"必须/禁止"（失去约束力）
