# guide

> 操作指南模板。文件名：`guide-<主题>-<行为>.md`

**区别于** standard：guide 是操作步骤（"打开 Y，加 Z 行"），standard 是约束规则（"必须用 kebab-case"）。

**必填节**：
- **前置条件** — 环境、权限、知识
- **操作步骤** — 逐步操作，每步有预期结果
- **预期结果** — 完成后的可验证状态

**可选节**：
- 排障 — 何时加：操作可能遇到常见错误
- 下一步/相关指南 — 何时加：有自然的后续阅读路径

**常见错误**：
- ❌ 步骤写成"配置 X"但没有说怎么配置
- ✅ 步骤写成"打开 Y 文件，添加 Z 行，预期看到 W"
- ❌ 没有前置条件（读者不知道需要什么环境）
