# changelog

> 变更日志模板。文件名：`changelog-<主题>-<行为>.md`

**必填节**：
- **版本头** — `## [version] - YYYY-MM-DD`
- **变更分类** — Added / Changed / Deprecated / Removed / Fixed / Security 按需出现

**可选节**：
- Unreleased — 未发布变更汇总
- YANKED — 已撤回版本标记

**常见错误**：
- ❌ 一个变更跨多个分类（选最主要的那个）
- ❌ 缺少 Security 分类（安全修复必须单独标注）
