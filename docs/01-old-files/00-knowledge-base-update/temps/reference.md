# reference

> 参考文档模板。文件名：`reference-<主题>-<行为>.md`

**区别于** api：reference 是纯数据查询（查配置项、查术语），api 是操作指引（怎么调接口）。

**必填节**：
- **概述** — 覆盖范围和约定
- **条目** — 按逻辑分组，每条含名称、描述、参数/类型/默认值（适用时）、示例（参考文档核心价值是可抄）

**可选节**：
- 版本历史 — 何时加：条目随版本变化
- 废弃标记 — 何时加：有计划移除的条目

**常见错误**：
- ❌ 条目没有示例（参考文档的核心价值是可抄）
- ❌ 概述写"本文档是 X 的参考"（循环定义）
